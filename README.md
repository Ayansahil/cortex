# Cortex — Your Second Brain 🧠

Save anything from the internet. Cortex automatically organizes, tags, and resurfaces it.

## What You Can Save
- 🔗 Articles & websites
- 🎬 YouTube videos
- 🐦 Tweets / X posts
- 📄 PDFs
- 🖼️ Images

## Features

**Auto Organization**
Items are automatically tagged and clustered into topics using AI (Google Gemma via OpenRouter) when saved.

**Semantic Search**
Search across everything you've saved using Gemini embeddings — finds results by meaning, not just keywords.

**Knowledge Graph**
Visual D3.js graph showing how your saved items connect through shared tags and topics.

**Collections**
Group related items into named collections with custom colors and emojis.

**Highlights**
Select any text inside a saved article to create color-coded highlights with personal notes.

**Memory Resurfacing**
Rediscovers things you saved weeks ago and brings them back to your dashboard.

**Related Items**
Each item shows semantically similar content from your library.

## Tech
React · Node.js · MongoDB · Redis · BullMQ · D3.js · Gemini API · OpenRouter
